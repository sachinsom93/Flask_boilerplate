### Flask Boiler Plate

### Technology stack
1. Flask
2. Flask-SQLALchemy
3. Pytest
4. Black
5. Flake
6. Tom

### SetUp Commans
1. Clone the repository
```sh
git clone https://github.com/sachinsom93/Flask_boilerplate.git
```

2. Install the dependencies
```sh
pip install -e .
```
 
3. Run the flask server
```sh
flask run
```
